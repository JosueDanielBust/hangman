# Hangman
---

Hangman game version made on NodeJS usign AngularJS and Firebase
With an expandible word core usign the `/add` path

## Instalation
Install the package dependencies using `npm install`

### Dependencies
* body-parser => ^1.15.2
* cookie-parser => ^1.4.3
* cookie-session => ^2.0.0-alpha.2
* express => 3.x
* firebase => ^3.6.1
* pug => ^2.0.0-beta6

## Running
Run using `npm start`
